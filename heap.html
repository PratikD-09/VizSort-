Heap Sort

Description

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure to sort elements.


It divides its input into a sorted and an unsorted region, and it iteratively shrinks the unsorted region by extracting the maximum element from it and moving that to the sorted region.

Complexity

Average Complexity O(n log n)
Best Case O(n log n)
Worst Case O(n log n)
Space Complexity O(1)
