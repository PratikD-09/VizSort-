Incertion Sort

Description

Insertion Sort is an iterative sorting algorithm that builds the final sorted array one element at a time. It's like sorting playing cards in your hand by shifting cards to their correct positions.


At each iteration, the algorithm takes one element from the unsorted part of the array and places it in its correct position in the sorted part. It does this by comparing the element with the previous elements in the sorted part and shifting them to the right until finding the correct position.


While not as fast as more advanced sorting algorithms like Quick Sort or Merge Sort, Insertion Sort can be more efficient for small datasets or nearly sorted arrays due to its simplicity. Average and worst-case time complexity is O(n^2), while the best-case time complexity is O(n).


Insertion Sort has some advantages over other quadratic sorting algorithms like Bubble Sort or Selection Sort, as it performs better on partially sorted arrays. However, it's less efficient for large datasets.

Complexity

Average Complexity O(n^2)
Best Case O(n)
Worst Case O(n^2)
Space Complexity O(1)
